## 🔥 Featured Projects

### 🎬 [Movie Recommender System](https://github.com/Atulkumar843/movie-recommender-system-tmdb-dataset-main.git)
Engineered an advanced content-based recommendation engine utilizing NLP vectorization and cosine similarity on metadata (genres, cast, keywords) for 5,000+ movies. Successfully deployed on Heroku, delivering real-time intelligent suggestions with a rapid **sub-1.5-second latency**.  
**Key Tech:** **Python** • **Scikit-Learn** • **Pandas** • **NLP Vectorization** • **Heroku Cloud Deployment**

### 🤖 [AI-Powered Real-Time Facial Emotion Detection](https://github.com/Atulkumar843/AI-Powered-Real-Time-Facial-Emotion-Detection.git)
Architected a live computer vision application capable of instantly classifying human facial emotions from video streams. Designed custom preprocessing pipelines to optimize frame processing rates and maintain high predictive reliability across dynamic lighting environments.  
**Key Tech:** **Python** • **Computer Vision** • **OpenCV** • **Machine Learning Classifiers** • **Real-Time Video Processing**

### 📧 [Spam SMS & Email Classifier](https://github.com/Atulkumar843/Spam-Email-SMS-Classifier.git)
Developed a high-precision Natural Language Processing classification model trained and validated on a robust dataset of 8,000+ textual records. Implemented advanced text tokenization and noise reduction, reducing false positives by **15%** while maintaining a **98% overall classification accuracy**.  
**Key Tech:** **Python** • **Natural Language Processing (NLP)** • **Scikit-Learn** • **Text Preprocessing** • **Statistical Modeling**

### 📲 [QR Code & Text-to-Voice Generator](https://github.com/Atulkumar843/QR-generator.git)
Programmed a high-concurrency, responsive utility application combining instant custom QR code generation with a real-time speech synthesis engine. Built utilizing modern Web APIs, successfully processing **500+ concurrent user inputs** with zero latency or execution errors.  
**Key Tech:** **HTML5** • **CSS3** • **JavaScript (ES6)** • **Web Speech API** • **QR Code API**
